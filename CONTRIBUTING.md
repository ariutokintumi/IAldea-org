# Contributing to IAldea

This repository follows the architecture and civic-safety boundaries described in [README.md](README.md) and [ialdea.org/manifesto](https://ialdea.org/manifesto).

- **Pre-build / sprint:** expect rapid structure changes until the MVP window closes; coordinate on the Pop-Up City channel before large refactors.
- **Pull requests:** every feature must be defensible against the *will not do* list in the README. Prefer small, focused PRs.
- **Upstream:** the canonical tree is [github.com/ariutokintumi/IAldea-org](https://github.com/ariutokintumi/IAldea-org); this fork may carry sprint-specific planning under `docs/planning/`.

Code of conduct: [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md). Security: [SECURITY.md](SECURITY.md).
