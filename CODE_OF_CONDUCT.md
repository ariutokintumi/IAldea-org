# Code of Conduct

We adopt the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1 as our community standard.

**Report concerns:** use project channels listed on [ialdea.org](https://ialdea.org/) or, for security-sensitive matters, see [SECURITY.md](SECURITY.md).

Maintainers may remove contributions (code, comments, issues) that violate this code or the civic-safety posture of IAldea (e.g. harassment, doxxing, or pressure to bypass safety limits).
